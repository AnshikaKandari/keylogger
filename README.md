# 🔍 Keylogger / Spyware Detector

A simple **GUI-based tool** built with Python and Tkinter to detect suspicious processes (like keyloggers, spyware, or malware) running on your system.  
It scans all running processes and highlights those containing suspicious keywords.

---

## 🚀 Features
- ✅ GUI built with **Tkinter**
- ✅ Scans all running processes using **psutil**
- ✅ Detects processes with suspicious keywords (like `keylog`, `spy`, `logger`, etc.)
- ✅ Highlights suspicious processes in **red**
- ✅ Alerts the user with a warning popup when suspicious activity is found

---

## 🛠️ Technologies Used
- **Python 3**
- **Tkinter** → GUI
- **psutil** → Process scanning
